## 📊 技术栈总览

### 前端
- **框架**: React 19 + TypeScript
- **UI库**: Ant Design 5.x
- **状态管理**: Zustand
- **图表**: ECharts
- **构建工具**: Vite

### 后端
- **数据库**: Supabase (PostgreSQL)
- **实时通信**: Supabase Realtime
- **认证**: Supabase Auth
- **存储**: Supabase Storage

### DevOps
- **容器化**: Docker
- **编排**: Docker Compose
- **CI/CD**: GitHub Actions
- **监控**: Prometheus + Grafana
- **日志**: Loki + Promtail
- **缓存**: Redis
